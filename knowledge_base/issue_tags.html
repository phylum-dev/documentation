Issue Tags

Phylum uses tags to uniquely identify issues raised by the system. These tags can be used in custom policy creation.

  • Severities
    • Minimum (I)
    • Medium (M)
    • High (H)
    • Critical (C)
    • Situation-dependent severity is represented in the table by .
  • Domains
    • Author Risk (A)
    • Engineering Risk (E)
    • Malicious Code (M)
    • Vulnerabilities (V)
    • License Risk (L)
