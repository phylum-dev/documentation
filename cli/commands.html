Command Line Tool

📄️ phylum

Client interface to the Phylum system

📄️ phylum analyze

Submit a request for analysis to the processing system

📄️ phylum auth

Manage authentication, registration, and API keys

📄️ phylum auth create-token

Create a new API token

📄️ phylum auth list-tokens

List all tokens associated with the logged-in user

📄️ phylum auth login

Login to an existing account

📄️ phylum auth register

Register a new account

📄️ phylum auth revoke-token

Revoke an API token

📄️ phylum auth set-token

Set the current authentication token

📄️ phylum auth status

Return the current authentication status

📄️ phylum auth token

Return the current authentication token

📄️ phylum exception

Manage analysis exceptions

📄️ phylum exception add

Add a new analysis exception

📄️ phylum exception list

List active analysis exceptions

📄️ phylum exception remove

Remove an existing analysis exception

📄️ phylum extension

Manage extensions

📄️ phylum extension install

Install extension

📄️ phylum extension list

List installed extensions

📄️ phylum extension new

Create a new extension

📄️ phylum extension run

Run an extension from a directory

📄️ phylum extension uninstall

Uninstall extension

📄️ phylum firewall

Manage the package firewall

📄️ phylum firewall log

Show firewall activity log

📄️ phylum group

Interact with user groups

📄️ phylum group create

Create a new group

📄️ phylum group delete

Delete a group

📄️ phylum group list

List all groups the user is a member of

📄️ phylum group member

Manage group members

📄️ phylum group member add

Add user to group

📄️ phylum group member list

List group members

📄️ phylum group member remove

Remove user from group

📄️ phylum history

Return information about historical jobs

📄️ phylum init

Setup a new Phylum project

📄️ phylum org

Manage organizations

📄️ phylum org link

Select an organization as default for all operations

📄️ phylum org list

List all organizations the user is a member of

📄️ phylum org member

Manage organization members

📄️ phylum org member add

Add user to organization

📄️ phylum org member list

List organization members

📄️ phylum org member remove

Remove user from organization

📄️ phylum org unlink

Clear the configured default organization

📄️ phylum package

Retrieve the details of a specific package

📄️ phylum parse

Parse dependency files and output their packages as JSON

📄️ phylum ping

Ping the remote system to verify it is available

📄️ phylum project

Manage Phylum projects

📄️ phylum project create

Create a new project

📄️ phylum project delete

Delete a project

📄️ phylum project link

Link a repository to a project

📄️ phylum project list

List all existing projects

📄️ phylum project status

Get current project information

📄️ phylum project update

Update a project

📄️ phylum status

Get Phylum project details

📄️ phylum uninstall

Uninstall the Phylum CLI

📄️ phylum update

Update to the latest release of the Phylum CLI

📄️ phylum version

Display application version
